---
name: git-hygiene
description: Use when inspecting or preparing Ronan's Flag Game git state safely.
---

- Confirm repo root, branch, and git status before edits.
- Keep the worktree clean of accidental unrelated edits.
- Do not commit, push, or deploy unless explicitly instructed.
- Do not rewrite history or discard user changes.
- Report files changed, validation run, and final git status.
