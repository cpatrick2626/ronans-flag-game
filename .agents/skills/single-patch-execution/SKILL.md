---
name: single-patch-execution
description: Use when a Ronan's Flag Game change should be made in one minimal patch with no unrelated refactors.
---

- Confirm repo root, branch, and git status before edits.
- Make the smallest safe change.
- Keep the write set narrow and explicit.
- Do not refactor unrelated code.
- Do not change gameplay logic unless explicitly authorized.
- Report files changed and final git status.
